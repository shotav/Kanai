<h1 align="center">Kanai</h1>
<h2 align="center">A lightweight cross-platform OpenGL text editor.</h2>

## Features

- [x] OpenGL
- [x] Cross-Platform
- [x] Portable
- [ ] Syntax Highlighting
- [ ] Code Completion
- [ ] File Hierarchy
- [ ] Terminal Emulator
- [ ] Auto Updater
- [ ] Settings Sync
- [ ] Lua Plugins

## Download

[Releases](https://github.com/PryosCode/Kanai/releases)

## Compile

```bash
git clone --recursive https://github.com/PryosCode/Kanai.git
cd Kanai
make
```
